\hmsection{System Implementation}{OAH}{UMA}\label{cha:implementation}

This chapter describes the implementation of the complete system, covering the image
processing pipeline, object detection, coordinate transformation, and hardware
integration. The system is built around three main components: a camera-based vision
module responsible for detecting and classifying balls, a Raspberry Pi~5 that
executes the processing pipeline and coordinates communication, and a robotic arm
that performs the physical sorting task. Each component is described in the
subsections below.
