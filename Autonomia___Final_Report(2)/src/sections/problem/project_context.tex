\hmsubsection{Project Context}{OAH}{UMA}

This project was assigned by the University of South-Eastern Norway (USN) and
constitutes a continuation of a bachelor project carried out in 2025. The previous
group developed a robotic sorting system based on the same hardware platform, but
was unable to complete a fully autonomous pick-and-place cycle before the end of
their project period. The present project therefore builds on their foundation
while addressing the limitations they reported, in particular insufficient inverse
kinematics accuracy and an unstable power architecture.

The assignment specifies that the system shall autonomously identify red and blue
spherical objects, pick them up, and sort them into separate containers. An
additional requirement states that the robot shall be designed so that it can be
mounted on a radio-controlled vehicle. For the present project, the focus is on
achieving reliable autonomous sorting on a stationary bench, with the mobile
integration considered a future extension.
