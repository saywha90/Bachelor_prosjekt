\hmsection{Domain}{OAH}{}

Robotic systems combined with computer vision form the main domain of this project. A
robotic system can be defined as a system capable of interacting with its environment through
sensing, processing, and actuation. In this report, the term "robot" refers to an autonomous
system that is able to detect, classify, and manipulate objects based on visual input.

This section will briefly explain a number of concepts relevant to robotics and computer
vision. This includes how robots perceive their environment using sensors, how image processing
techniques can be used to detect and classify objects, and how embedded systems enable
communication and control between different components. Additionally, an overview of how
such systems operate in real-time and under physical constraints will be provided.

The concepts presented in this section form the foundation for the development of a robotic
system capable of detecting and sorting coloured objects.
