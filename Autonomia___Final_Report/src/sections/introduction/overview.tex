\hmsubsection{Overview}{OAH}{}
This project concerns the development of a robotic system for automated
sorting of objects based on colour. The system shall be capable of
identifying and distinguishing between red and blue balls and subsequently
placing them into two separate containers such that each container contains
only balls of a single specified colour.

To achieve this, the system must integrate multiple technical disciplines, including image processing, software development, and mechanical design. The robot is equipped with a camera mounted at an elevated position, which is used to capture and analyse visual data from its surroundings. Based on this analysis, the system shall be able to classify objects according to colour and execute the necessary actions to perform the sorting task.

Additionally, the system is required to be mountable on a radio-controlled car, which introduces constraints related to size, weight, stability, and the integration of mechanical and electronic components.

The project is carried out by a multidisciplinary team consisting of four computer engineering students and one mechanical engineering student, reflecting the need for both software and hardware expertise in the development of the system.
